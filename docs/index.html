What the $1M calculator does

Estimate how long it could take to save $1M.

The million dollar calculator estimates a broad path from today's money situation to $1,000,000. It uses ranges for income, assets, debt pressure, yearly investing, savings growth, and expected returns, then sends users to a playful Money MBTI-style result on the official DecisionKit page.
