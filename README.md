## How long will it take to save $1 million?

The DecisionKit million dollar calculator is for people who want a broad estimate of the distance between today and $1,000,000. It uses ranges instead of precise account-level inputs, then opens the official live calculator on DecisionKit.

## What it estimates

- A broad timeline range for reaching $1,000,000.
- How current assets, investing habits, debt pressure, and expected returns can change the estimate.
- A shareable Money MBTI-style result that makes the output easier to compare and discuss.
